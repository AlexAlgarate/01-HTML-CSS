My projects

Strava analysis
Proyecto que usa la API de Strava para obtener datos y mostrarlos por consola. Sirve para ver resúmenes semanales de las actividades realizadas, ver más detalles de esas actividades, ver los streams de las actividades de la semana en curso, o exportar esos streams en formato 'csv' para poder ser usados en algún otro programa de análisis deportivo.

Zaidin Coffee Scraper

Scraper web creado en Python para obtener el catálogo de café de una web de café de especialidad. El código sigue una arquitectura hexagonal, principios SOLID y de Clean Architecture. El formato de presentación de los datos es Markdown para una fácil visualización de la información.

Roman Calculator Tkinter App

Proyecto del módulo de Ingenieria de Software del bootcamp de Keepcoding donde creamos una calculadora de números arábigos a números romanos. Creada conla librería Tkinter, se puede desplegar en cualquier entorno de escritorio. Proyecto completo donde se aplica el Modelo-Vista-Controlador (MVC) cubierto con suit de tests creada con Pytest.

Wedding web template

La página web de mi boda creada con el framework de Python llamado Reflex. Desplegada en Vercel, diseño responsive creada siguiendo el criterio de mobile first. La versión con backend usa un formulario que guarda la información en una base de datos alojada en Supabase, el backend está desplegado en un contenedor de Docker alojado en Railway (ver proyecto aquí).

Swimming Pace Calculator

Calculadora creada con Tkinter para calcular el ritmo medio de natación. El usuario rellena los campos de distancia y tiempo y automáticamente se muestra el ritmo medio (medido en minutos:segundos/100m) para cubrir esa distancia en ese tiempo nadando.
